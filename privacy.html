Privacy Policy

Last updated: May 1, 2025

KUON Music Tools（以下「本ツール」）は、ユーザーのプライバシーを尊重します。本ページでは、本ツールが収集・使用する情報について説明します。

This Privacy Policy explains how KUON Music Tools handles information when you use our tool to post content to TikTok via the TikTok Content Posting API.

1. 収集する情報 / Information We Collect

本ツールは以下の情報を取り扱います:

  • TikTok アクセストークン: TikTok の OAuth 2.0 認証によって発行されるトークン。動画投稿のために使用します。
  • TikTok クライアント情報: Developer ポータルで発行された Client Key および Client Secret。

The tool handles the following information:

  • TikTok Access Token: Issued via TikTok's OAuth 2.0 flow. Used solely to post videos to your TikTok account.
  • TikTok Client Credentials: Client Key and Client Secret issued from the TikTok Developer Portal.

2. 情報の保存場所 / Where Information Is Stored

すべての認証情報は、ユーザー自身のローカルPCにのみ保存されます。外部サーバーやクラウドへのアップロードは行いません。

All credentials and tokens are stored locally on the user's own computer only. No data is uploaded to any external server or cloud service.

  • うたまち用トークン: tiktok_token.json（ローカル）
  • Grace Notes用トークン: tiktok_token_gn.json（ローカル）

3. 情報の使用目的 / How We Use Information

取得した情報は、以下の目的のみに使用します:

  • TikTok への動画コンテンツの投稿
  • 投稿ステータスの確認

Collected information is used solely for:

  • Posting video content to TikTok on behalf of the authenticated account
  • Checking the publish status of uploaded videos

4. 第三者への情報提供 / Third-Party Disclosure

本ツールは、いかなる第三者にも個人情報・認証情報を提供・販売・共有しません。

We do not sell, trade, or transfer any user information or credentials to third parties.

5. TikTok API の使用 / Use of TikTok API

本ツールは TikTok Content Posting API を使用します。TikTok 自身のプライバシーポリシーは こちら をご参照ください。

This tool uses the TikTok Content Posting API. TikTok's own Privacy Policy can be found at TikTok Privacy Policy.

6. データの削除 / Data Deletion

認証情報を削除するには、ローカルPCに保存された以下のファイルを削除してください:

  • tiktok_token.json
  • tiktok_token_gn.json
  • tiktok_client.json
  • tiktok_client_gn.json

To delete your data, simply remove the above files from your local computer.

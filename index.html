KUON Music Tools は、オリジナル音楽コンテンツの制作から SNS 配信までを自動化する、クリエイター向けのツールセットです。

🎵

うたまち

日本各地の地名をテーマにした音楽コンテンツを自動生成し、YouTube・TikTok へ配信します。

✝️

Grace Notes

クリスチャン向けのインストゥルメンタル・ワーシップ音楽を自動生成し、各 SNS プラットフォームへ配信します。

📲

TikTok 連携

TikTok Content Posting API を使用して、生成した動画コンテンツをアカウントに自動投稿します。

このツールについて / About This Tool

KUON Music Tools は、個人クリエイターが運営する音楽コンテンツ自動化ツールです。

TikTok API を使用して行う操作:

  • 生成した短編動画（Shorts）を、認証済みのTikTokアカウントに投稿する
  • 投稿ステータスを確認する

このツールはユーザー自身のPCで動作し、第三者のサーバーにデータを送信しません。
